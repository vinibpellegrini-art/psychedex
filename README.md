# Psychedex

Psychedex is a personal project built to explore and organize information about pharmaceutical substances in a simple and structured way.

## 🚀 Tech Stack

- React
- TypeScript
- Vite
- Supabase (PostgreSQL)

## 📦 Features

- List and organize pharmaceutical substances
- Data stored in a PostgreSQL database via Supabase
- Search and category filtering
- Drug interaction checker between selected substances
- Automatic fetching of molecular structures via PubChem API
- Neon-dark UI with color-coded categories and an interactive background

## 🔧 Planned Features

- User authentication and accounts
- Per-substance dosage and harm-reduction information
- Improved database structure and scalability

## 💡 About the project

This project was built mainly as a way to practice and improve my development skills, experimenting with frontend architecture, API integration, and database design.
