# Psychedex

Psychedex is a personal project built to explore and organize information about pharmaceutical substances in a simple and structured way.

## 🚀 Tech Stack

- React
- TypeScript
- Vite

## 📦 Features

- List and organize pharmaceutical substances
- Structured data using TypeScript
- Basic search and navigation system
- Automatic fetching of molecular structures via PubChem API

## 🔧 Planned Features

- C# backend with authentication system
- Drug interaction checker between substances
- Improved database structure and scalability

## 💡 About the project

This project was built mainly as a way to practice and improve my development skills, experimenting with frontend architecture, API integration, and future backend expansion.
